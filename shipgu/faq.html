❓ 자주묻는질문

SHIPGU 서비스 이용 중 자주 묻는 질문들을 모아놓았습니다

  • Q
    QR코드 결제시 오류가 발생했어요. 어떻게 해야 하나요?
    ▼

    QR코드 결제 오류가 발생하는 경우 다음과 같은 원인들이 있을 수 있습니다:

    주요 해결 방법:
    1. QR코드 유효시간(5분) 확인 후 새로 생성
    2. 결제 앱의 최신 버전 확인
    3. 네트워크 연결 상태 점검
    4. 잔액 부족 여부 확인

    위 방법으로도 해결되지 않으면 1:1 문의를 통해 고객센터로 연락해 주시기 바랍니다.

  • Q
    지원하는 결제 수단이 무엇인가요?
    ▼

    SHIPGU에서는 다음 결제 수단을 지원합니다:

    • 알리페이 (Alipay) - 중국 최대 모바일 결제 플랫폼
    • 위챗페이 (WeChat Pay) - 위챗 기반 간편 결제
    • 웨이디엔 (UnionPay) - 중국 은련 카드 결제

    모든 결제 수단은 실시간 환율이 적용되며, 개인회원도 이용 가능합니다.

  • Q
    개인회원과 사업자회원의 차이점이 무엇인가요?
    ▼

    회원 유형에 따라 이용할 수 있는 서비스가 다릅니다:

    개인회원:
    • QR코드 결제시스템만 이용 가능
    • 사업자번호 등록 불필요

    사업자/법인회원:
    • QR코드 결제시스템
    • 1688솔루션 (수수료 없는 직접 구매)
    • 무역송금(T/T) 서비스
    • 사업자번호 등록 필요
  • Q
    사업자회원으로 전환하려면 어떤 서류가 필요한가요?
    ▼

    사업자회원 전환을 위해서는 다음 서류가 필요합니다:

    필수 서류:
    • 사업자등록증 사본
    • 대표자 신분증 사본
    • 통장 사본 (사업자명 확인용)

    추가 서류 (법인의 경우):
    • 법인등기부등본
    • 인감증명서

    서류 제출 후 1-2 영업일 내에 승인 결과를 알려드립니다.

  • Q
    환율은 어떻게 적용되나요?
    ▼

    SHIPGU는 실시간 환율 시스템을 사용합니다:

    • 실시간 시장 환율 + 7원의 적용환율 사용
    • 30초마다 자동 업데이트
    • 결제 시점의 환율이 최종 적용
    • 좌측 사이드바에서 현재 적용환율 확인 가능
  • Q
    로그인이 안 되거나 화면이 제대로 표시되지 않아요.
    ▼

    기술적 문제 해결을 위한 단계별 방법:

    1단계: 브라우저 새로고침 (Ctrl+F5)
    2단계: 브라우저 캐시 및 쿠키 삭제
    3단계: 다른 브라우저에서 접속 시도
    4단계: 인터넷 연결 상태 확인

    위 방법으로 해결되지 않으면 사용 중인 브라우저와 기기 정보를 포함하여 1:1 문의해 주세요.

  • Q
    결제 내역은 어디서 확인할 수 있나요?
    ▼

    결제 내역 확인 방법:

    • 좌측 사이드바 > 계정 관리 > 결제내역 메뉴
    • 모든 QR코드 결제 기록 확인 가능
    • 날짜별, 금액별 검색 기능 제공
    • 영수증 다운로드 및 재발행 가능
